# Template for quick prototyping

### Prerequest

1. node
2. yarn
3. python

#### mac

best by installing with [**HomeBrew**](https://brew.sh/)

```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

`brew install node`

`brew install yarn`

### Install

`yarn install`

## Use

`yarn start`
